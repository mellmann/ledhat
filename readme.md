## I2C - Qwiic

All Qwiic cables have the following color scheme and arrangement:

  * Black = GND
  * Red = 3.3V
  * Blue = SDA
  * Yellow = SCL
  
ESP32

* D22 (GPIO22) = SCL
* D21 (GPIO21) = SDA
